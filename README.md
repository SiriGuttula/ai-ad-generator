\# AI Advertisement Generator



A full-stack AI-powered application that generates marketing advertisements automatically.



\## Technologies Used



Frontend:

\- Angular

\- HTML

\- CSS

\- TypeScript



Backend:

\- ASP.NET Core Web API

\- C#



AI Integration:

\- OpenAI API



\## Features



\- Generate marketing advertisements using AI

\- Angular frontend user interface

\- .NET Web API backend

\- Integration with OpenAI for ad generation
